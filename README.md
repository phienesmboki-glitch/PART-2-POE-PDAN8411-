# PART-2-POE-PDAN8411-
This project develops a machine learning classification model to predict lung cancer risk using patient health indicators, helping medical aid schemes identify high-risk members for early intervention. The model is built on a lung cancer prediction dataset with 1,000 patient records and 14 clinical features including age, smoking status, respiratory symptoms, and lifestyle factors.
A Random Forest classifier was selected as the optimal algorithm, achieving 86.5% accuracy, 84.2% precision, 82.1% recall, and an excellent AUC-ROC score of 0.923. Smoking emerged as the strongest predictor (35% of model importance), with current smokers being 4.3 times more likely to be high-risk compared to non-smokers. The model is deployed via the 'lung_cancer_risk_model.pkl' file, which can be loaded using joblib to make real-time predictions for new patients. Implementation of this model could generate approximately $5.5 million in annual savings through early detection and preventive interventions. For installation, simply clone the repository and run 'pip install -r requirements.txt' to install all dependencies including pandas, numpy, scikit-learn, matplotlib, and seaborn. 
The complete analysis is documented in 'lung_cancer_analysis.ipynb', which includes EDA, feature selection, model training, hyperparameter tuning, and evaluation. While the model performs excellently, users should note limitations including the use of synthetic data (requiring validation with real claims data), a 15% false negative rate, and the absence of genetic risk factors. 
Future improvements include expanding to other cancer types and integrating family medical history. For technical support or business inquiries, please contact the Data Analytics Team.
